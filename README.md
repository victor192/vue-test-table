# vue-test
Example of a dynamic table in Vue that can be filtered and sorted.

Tools used:
* [Vue.js](https://github.com/vuejs/vue)
* [Vuex](https://github.com/vuejs/vuex)
* [Axios](https://github.com/axios/axios)
* ES6 Features
* SCSS
* BEM naming

## Project setup
```
npm install
```
